# ビル管理士 週次過去問トレーニング

## 概要
建築物環境衛生管理技術者（ビル管理士）試験対策の学習ツール。
ブラウザで開くと、その週7日分（1日21問×7日＝147問）の過去問が日別に表示される。

## 公開先
- GitHub: https://github.com/shimizu-eu/my-app
- GitHub Pages: https://shimizu-eu.github.io/my-app/
- PC・スマホ両対応

## ファイル構成
```
bldg-exam/
  index.html    — 単一HTMLファイル（問題データ・ロジック・UIすべて含む）
  CLAUDE.md     — このファイル
```

## 技術仕様
- 単一HTMLファイル（外部ライブラリなし、バニラJS）
- localStorage でブラウザに回答履歴・統計を保存
- 日付ベースのシード付き乱数で週ごとに異なる問題を自動選出
- goal-tracker.html と同じCSS変数・デザインパターンを使用

## よくある依頼と対応方法

### 問題を追加したい
1. index.html の `QUESTION_BANK` 内の該当章の `questions` 配列に追加
2. id は「章番号-連番」（例: "3-011"）で採番
3. answer は 0始まりのインデックス
4. 追加後 `git add . && git commit && git push` で公開に反映

### 問題を修正したい
1. 該当の id で index.html 内を検索
2. text / choices / answer / explanation を修正
3. push で反映

### UIを変更したい
- CSS は `<style>` タグ内にすべて記載
- JS は `<script>` タグ内にすべて記載
